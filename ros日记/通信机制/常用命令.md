### 1.指令
```shell {.line-numbers}
rosnode // node 信息
rostopic // 话题信息
rosservice //服务信息
rosmsg //消息信息
rossrv //两端消息信息
rosparam //参数信息
```


### 2.指令二级参数
```
rosnode ping    测试到节点的连接状态
rosnode list    列出活动节点
rosnode info    打印节点信息
rosnode machine    列出指定设备上节点
rosnode kill    杀死某个节点
rosnode cleanup    清除不可连接的节点
```
